# lib/brevo/

Helpers Brevo spécifiques (gestion newsletter à 3 tags, listes, segmentation).

L'envoi transactionnel passe par `lib/email/` qui suit le pattern adapter avec mock. Ce dossier accueillera les fonctions Brevo propres au mouvement (newsletter taggée par origine + action + département).

Implémentation au **chantier 1.2** (validation email) et **8.1** (newsletter).
